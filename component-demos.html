Component Library

Interactive demos of all visualization components. Click a component in the sidebar to jump to its demo.

Sparkline

Mini trend visualization for inline display. Shows data trends in a compact format.

Upward trend
Downward trend
Volatile
With area fill 
const sparkline = new Sparkline('container', {
  width: 100,
  height: 30,
  color: 'var(--accent-primary)',
  showDot: true,
  showArea: false
});
sparkline.update([10, 15, 12, 18, 25, 22, 30]);

Network Graph

Force-directed network visualization for people and organizations. Drag nodes to rearrange.

const network = new NetworkGraph('container', {
  height: 400,
  nodeRadius: 20,
  linkDistance: 120,
  onNodeClick: (node) => console.log(node)
});
network.update({ nodes: [...], links: [...] });

Stacked Area Chart

Volume over time by faction visualization. Shows cumulative activity across multiple groups.

const chart = new StackedAreaChart('container', {
  height: 280
});
chart.update({
  dates: ['2025-01-01', '2025-01-02', ...],
  series: [[10, 20, ...], [15, 25, ...]],

});

Timeline + Volume Composite

Combined visualization showing volume over time with event markers. Correlate mention spikes with specific events. Toggle between faction and publisher views. Synchronized zoom and cross-highlighting between charts.

const composite = new TimelineVolumeComposite('container', {
  height: 450,
  volumeHeight: 180,
  timelineHeight: 180,
  showViewToggle: true,  // Enable faction/publisher toggle
  onEventClick: (event) => console.log(event)
});
composite.update({
  volumeData: {  // Volume by faction
    dates: ['2025-01-10', '2025-01-11', ...],
    series: [[100, 150, ...], [80, 120, ...]],
    factions: [{ id: 'f1', name: 'State Actors', color: '#ef4444' }]
  },
  publisherData: {  // Volume by publisher (enables toggle)
    dates: ['2025-01-10', '2025-01-11', ...],
    series: [[80, 120, ...], [60, 90, ...]],
    publishers: [{ id: 's1', name: 'Twitter', color: '#000' }]
  },
  events: [
    { id: '1', text: 'Initial article', date: '2025-01-10T08:00:00Z' },
    { id: '2', text: 'Viral spread', date: '2025-01-12T14:30:00Z' }
  ]
});

  showStatus: true,
  showSparkline: true,
  showVolume: true,
  showSubNarratives: true,      // Enable expandable themes
  maxSubNarratives: 5,           // Max themes shown when expanded
  defaultShowDescription: false, // Show descriptions by default
  onNarrativeClick: (narrative) => console.log(narrative)
});
list.update({
  narratives: [
    { 
      id: '1', 
      text: 'Election fraud claims', 
      sentiment: -0.68, 
      status: 'new',
      factionMentions: { f1: { volume: 12500 } },
      volumeOverTime: [...]
    }
  ]
});

Document Table

Flexible table view for documents with customizable columns. Supports publisher, title, excerpt, date, and related entities (narratives, themes, events, locations, people, organizations, factions, topics). Click column headers to sort.

Filter columns: 
// Document Table with customizable columns
const table = new DocumentTable('container', {
  columns: ['publisherName', 'title', 'publishedDate', 'narratives'],
  maxItems: 50,
  sortable: true,
  defaultSort: 'publishedDate',
  defaultSortDir: 'desc',
  onDocumentClick: (doc) => console.log(doc)
});
table.update({ documents: [...] });

// Column Filter component for interactive column selection
const columnFilter = new ColumnFilter('filter-container', {
  availableColumns: {
    publisherName: 'Publisher',
    title: 'Title',
    publishedDate: 'Published',
    narratives: 'Narratives',
    persons: 'People'
  },
  defaultColumns: ['publisherName', 'title', 'publishedDate'],
  requiredColumns: ['title'],
  onChange: (columns) => table.setColumns(columns)
});
columnFilter.render();

Time Range Filter

Histogram with brush selection for filtering data by time range. Drag to select a range, or click and drag the selection to move it.

All Time 
const filter = new TimeRangeFilter('container', {
  height: 48,
  barColor: 'var(--accent-primary)',
  onChange: (range) => {
    if (range) {
      console.log('Selected:', range.start, 'to', range.end);
    } else {
      console.log('Selection cleared');
    }
  }
});
filter.update({
  dates: ['2025-01-01', '2025-01-02', ...],
  volumes: [100, 150, 120, ...]
});

Document Content Renderer

Renders document content based on document type. Supports social posts, TikTok videos, news articles, and internal documents with highlights and comment annotations.

const renderer = new DocumentContentRenderer('container', {
  showPortionMarks: true,
  showAnnotations: true
});
renderer.update({
  document: {
    documentType: 'social_post',  // or 'news_article', 'tiktok', 'internal'
    author: { displayName: 'User', username: '@user' },
    content: 'Post content here...',
    publishedDate: '2025-01-15T10:30:00Z'
  },
  highlights: [...],  // Optional user highlights
  comments: [...]     // Optional comments
});

Topic List

List of topics with status indicators, sparklines, and expandable bullet point summaries. Shows topic duration and activity over time.

const list = new TopicList('container', {
  maxItems: 10,
  showSparkline: true,
  showVolume: true,
  showDuration: true,
  showBulletPoints: false,
  maxBulletPoints: 3,
  onTopicClick: (topic) => console.log(topic)
});
list.update({
  topics: [
    {
      id: '1',
      headline: 'Topic headline',
      startDate: '2025-01-01',
      endDate: null,  // null = active
      bulletPoints: ['Point 1', 'Point 2'],
      volumeOverTime: [{ date: '2025-01-01', volume: 100 }, ...]
    }
  ]
});
